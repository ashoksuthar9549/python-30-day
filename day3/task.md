# Day 3: Lists & Tuples
**Project/Task**: Create an inventory list manager script.

## Problem Statement
In `day03_inventory.py`, you will manage a simple list of items in an inventory. The script should do the following in order:
1. Create a variable called `inventory` that holds a list of 3 starting items (e.g., `["apple", "banana", "orange"]`).
2. Print the initial inventory to the console.
3. Use the `.append()` method to add a new item (like `"grape"`) to the list.
4. Use the `.remove()` method to remove one of the original items (like `"banana"`).
5. Print the updated inventory to the console.
6. Print the total number of items currently in the inventory using the `len()` function (e.g., `"Total items: 3"`).
