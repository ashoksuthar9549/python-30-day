# Day 5: Loops (for & while)
**Project/Task**: Build a CLI menu that loops until 'exit' is typed.

## Problem Statement
Create a Python script that uses a `while` loop to continuously display a menu until the user decides to quit. The program should:
1. Print a menu: `1. Say Hello | 2. Show time | 3. Exit`
2. Ask the user for their choice using `input()`.
3. If the user types `"1"`, print `"Hello there!"`.
4. If the user types `"2"`, print `"It is time to learn Python!"`.
5. If the user types `"3"` or `"exit"`, print `"Goodbye!"` and **break** out of the loop so the program ends.
6. If the user types anything else, print `"Invalid choice."` and let the loop run again.
