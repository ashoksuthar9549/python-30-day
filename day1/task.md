# Day 1: Variables & Data Types
**Project/Task**: Write a basic CLI calculator script.

## Problem Statement
Create a Python script that acts as a simple calculator. The program should:
1. Ask the user to input a first number.
2. Ask the user to input a mathematical operator (e.g., `+`, `-`, `*`, `/`).
3. Ask the user to input a second number.
4. Perform the requested calculation based on the given operator.
5. Print the result in a readable format (e.g., `Result: 5.0`).

*Note: The `input()` function in Python returns a string, so you'll need to convert the user's input into a numeric data type (like `float` or `int`) before doing the math.*
