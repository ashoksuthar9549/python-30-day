# Day 4: Dictionaries
**Project/Task**: Build a user profile dictionary generator.

## Problem Statement
Create a Python script that asks the user for 3 pieces of information and stores them in a Dictionary. The script should:
1. Ask the user to input their name, age, and city.
2. Create a dictionary (using curly braces `{}`) with keys `"name"`, `"age"`, and `"city"`, and assign the user's inputs to these keys.
3. Print the dictionary to the console.
4. Access and print just the `"city"` value from the dictionary using its key.
