# Day 2: Control Flow (if/elif/else)
**Project/Task**: Build a text-based guessing game.

## Problem Statement
Create a Python script where the computer "thinks" of a secret number, and the user has to guess it. The program should:
1. Define a hardcoded secret number (e.g., `secret_number = 7`) or generate one randomly using `random.randint()`.
2. Ask the user to input a guess.
3. Compare the user's guess to the secret number using `if/elif/else` statements.
4. If the guess is too high, print `"Too high! Try again."`
5. If the guess is too low, print `"Too low! Try again."`
6. If the guess is exactly correct, print `"Congratulations! You guessed it!"`
